Edit Mode dans Blender

Source de 09:00 à 11:48

Le Edit Mode dans Blender permet aux utilisateurs de modifier la géométrie des objets 3D en travaillant directement sur leurs vertices, arêtes et faces.

Voici comment utiliser le Edit Mode :

  • Sélection de l'Objet : Sélectionnez l'objet que vous souhaitez modifier.
  • Accès au Edit Mode : Appuyez sur la touche Tab pour basculer entre Object Mode et Edit Mode.
  • Modification de la Géométrie : Sélectionnez et manipulez les vertices, arêtes et faces avec les raccourcis clavier comme G (déplacer), R (rotation), S (mise à l'échelle) et E (extrusion).
  • Sélection de Surfaces : Utilisez Ctrl + clic pour sélectionner plusieurs surfaces à modifier.
  • Navigation et Utilisation : Utilisez la souris pour zoomer et dézoomer, et utilisez Shift + clic droit pour déplacer la vue.
