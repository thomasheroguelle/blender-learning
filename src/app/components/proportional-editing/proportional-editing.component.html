Édition Proportionnelle dans Blender

Source de 11:44 à 13:36

L'édition proportionnelle dans Blender permet d'appliquer des transformations de manière fluide et graduelle sur une sélection, en influençant les vertices, arêtes et faces avoisinantes. Cela est particulièrement utile pour créer des déformations organiques et des transitions douces sur des objets complexes.

Voici comment utiliser l'édition proportionnelle :

  • Activation de l'Édition Proportionnelle : Appuyez sur la touche O pour activer ou désactiver l'édition proportionnelle en mode Edit.
  • Sélection de l'Objet : Sélectionnez l'objet que vous souhaitez modifier.
  • Accès au Edit Mode : Appuyez sur la touche Tab pour basculer entre Object Mode et Edit Mode.
  • Utilisation : Utilisez les raccourcis clavier comme G (déplacer), R (rotation) et S (mise à l'échelle) pour manipuler les vertices, arêtes et faces.

Différence entre l'Édition Proportionnelle et Non Proportionnelle

Lorsque vous utilisez le raccourci Ctrl + G :

  • Sans Édition Proportionnelle : Les transformations affectent uniquement les éléments sélectionnés, sans influencer les vertices, arêtes et faces avoisinantes.
  • Avec Édition Proportionnelle : Les transformations appliquées sur les éléments sélectionnés influencent également les éléments avoisinants selon un rayon de proportionnalité défini, visible par un cercle d'influence. Cela permet des transitions plus douces et des déformations plus naturelles.
