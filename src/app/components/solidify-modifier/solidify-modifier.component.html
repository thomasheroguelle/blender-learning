Solidify Modifier dans Blender

Source de 05:06 à 06:30

Le Solidify Modifier est un outil dans Blender qui permet de donner de l'épaisseur à des surfaces ou des objets 3D. Cet outil est particulièrement utile pour transformer des surfaces bidimensionnelles (2D) en objets tridimensionnels (3D) avec une épaisseur définie.

Voici comment utiliser le Solidify Modifier :

  • Sélection de l'Objet : Sélectionnez l'objet auquel vous souhaitez ajouter de l'épaisseur.
  • Ajout du Modifier : Allez dans le panneau des modifiers et ajoutez le Solidify Modifier.
  • Ajustement de l'Épaisseur : Réglez le paramètre "Thickness" pour définir l'épaisseur souhaitée.
  • Autres Paramètres : Utilisez les paramètres comme "Offset", "Even Thickness", et "High Quality Normals" pour affiner l'effet.
  • Application du Modifier : Une fois satisfait, appliquez le modifier pour rendre l'épaisseur permanente.
