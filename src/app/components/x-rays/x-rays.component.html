Rayons X dans Blender

Source de 02:26 à 05:05

La fonctionnalité des Rayons X dans Blender permet aux utilisateurs de voir et de sélectionner les géométries internes d'un objet en rendant les objets semi-transparents. Cela facilite le travail sur des structures complexes en permettant une meilleure visualisation et manipulation des vertices, arêtes, et faces situés à l'intérieur d'un modèle 3D.

Comment Activer les Rayons X :

  • Sélection de l'Objet : Sélectionnez l'objet sur lequel vous souhaitez activer les Rayons X.
  • Accès au Mode Objet : Assurez-vous d'être en Object Mode en appuyant sur la touche Tab si nécessaire.
  • Activation des Rayons X : Dans la barre d'outils en haut de l'écran, cliquez sur l'icône de Rayons X (représentée par une double flèche croisée) ou appuyez sur Alt + Z pour activer ou désactiver les Rayons X.
  • Réglage de l'Opacité : Une fois les Rayons X activés, vous pouvez ajuster l'opacité dans le panneau des propriétés pour obtenir la transparence désirée.

Avantages des Rayons X :

L'utilisation des Rayons X offre plusieurs avantages pour la modélisation 3D :

  • Sélection Facile : Sélectionnez facilement les éléments internes sans avoir à masquer ou à découper l'objet.
  • Visualisation Améliorée : Obtenez une meilleure vue d'ensemble des structures internes complexes.
  • Manipulation Précise : Manipulez avec précision les vertices, arêtes, et faces internes sans interférence des géométries extérieures.
